# Contributing to Radar and LiDAR Fusion

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the fork** to your local machine.
3. **Create a new branch** for your feature or bug fix.
4. **Make your changes** and ensure they follow the project's coding style.
5. **Add tests** for any new functionality.
6. **Ensure all tests pass** (`cargo test` for Rust, run simulation for MATLAB).
7. **Commit your changes** with a descriptive commit message.
8. **Push the branch** to your fork.
9. **Submit a pull request** to the main repository.

## Coding Standards

### Rust
- Use `cargo fmt` to format your code.
- Ensure all tests pass with `cargo test`.
- Document new functions and structs.

### MATLAB
- Use clear variable names.
- Add comments to explain complex logic.
- Keep functions modular.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue on GitHub with a detailed description and steps to reproduce if applicable.
